# NDVX Systems
**A new perspective on creating:**
NDVX Systems is a software network that aims to revolutionize the way we create and interact with digital content. Why have text documents, presentations, spreadsheets and more as separate entities? Why not anything between them? Don't be limited to some pre-made templates. Create your own!

NDVX Systems will start as a simple console application, but the vision is to use NDVX lite-ui to create a more user-friendly interface in the future. The goal is to provide a flexible and powerful platform for creators of all kinds, allowing them to express their ideas in new and innovative ways.
